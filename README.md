# 🌲 迷雾森林 — 文字探险解谜

一款多 Agent 协作驱动的终端文字冒险解谜游戏。玩家在充满迷雾的森林中探索不同场景，破解谜题收集印记，最终通关或被迷雾吞噬。

## 安装

确保已安装 [Node.js](https://nodejs.org/)（v18+）。

```bash
git clone https://github.com/yinapan/misty-forest.git
cd misty-forest
npm install
```

## 启动

```bash
npm run build
npm start
```

也可以使用开发模式直接运行（无需编译）：

```bash
npm run dev
```

启动后游戏会显示教程说明并自动进入第一个场景，按照提示输入指令即可开始冒险。

## 游戏指令

| 指令 | 别名 | 说明 |
|------|------|------|
| 直接输入文字 | — | 回答当前谜题 |
| `探索` | `look` / `查看` / `观察` | 重新查看当前场景描述 |
| `提示` | `hint` | 获取当前谜题的提示（消耗 HP） |
| `放弃` | `skip` / `跳过` | 放弃当前谜题，进入下一场景（-20 HP） |
| `状态` | `status` | 查看 HP、印记、道具等信息 |
| `道具` | `item` / `使用 [道具名]` | 查看或使用道具 |
| `帮助` | `help` / `?` | 显示帮助信息 |
| `重新开始` | `restart` / `reset` | 重置游戏，从头开始 |
| `退出` | `quit` / `exit` | 退出游戏 |

## 玩法规则

### 血量（HP）

- 初始 HP：**100**，上限 100
- 答对谜题（无提示）：**+10 HP**
- 答对谜题（使用过提示）：**+5 HP**
- 答错谜题：第 1 次 **-10 HP**，第 2 次 **-15 HP**，第 3 次 **-20 HP**
- 请求提示：Level 1 **-5 HP**，Level 2 **-10 HP**，Level 3 **-15 HP**
- 放弃谜题：**-20 HP**

### 印记

- 每个场景对应一道谜题，答对即获得该场景的印记
- 每道谜题最多尝试 **3 次**，3 次均答错则该场景失败（不获得印记），自动进入下一场景
- 每道谜题最多可使用 **3 级提示**

### 通关条件

集齐 **5 枚**场景印记，游戏胜利通关。

### 失败条件

HP 降为 **0** 或以下，游戏结束。

### 道具系统

探险过程中有概率获得道具（背包上限 3 个）：

| 道具 | 效果 |
|------|------|
| 生命露珠 | 恢复 20 HP |
| 迷雾灯笼 | 下次请求提示免费（不扣 HP） |
| 时间沙漏 | 额外获得一次答题机会 |

使用方式：输入 `使用 生命露珠` 或 `道具 生命露珠`。
